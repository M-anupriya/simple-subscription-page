# simple-subscription-pag
Project Overview

The webpage displays three subscription plans:

Free Plan (0/-)

Standard Plan (499/-)

Premium Plan (699/-)

Each plan clearly shows the available features such as:

Listen to music

Share links

Create a playlist

Ad-free music

Download music

Upload music

The page is designed using a clean card layout and proper alignment to compare different subscription plans easily.

Features

Structured pricing cards

Feature comparison between plans

Clean and modern UI design

Responsive layout using Flexbox

Beginner-friendly project

Technologies Used

HTML5

CSS3

Flexbox

How to View the Project

Clone or download this repository.

Open the index.html file in your browser.
